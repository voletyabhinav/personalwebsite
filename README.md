# Abhinav Volety — Personal Website

Next.js personal site with React, Tailwind CSS, and shadcn/ui.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech

- **Next.js 14** (App Router)
- **React 18**
- **Tailwind CSS** for styling
- **shadcn/ui** (Button, Card) — Apple Garamond for titles/body, Lacquer for buttons

## Add more shadcn components

```bash
npx shadcn@latest add <component-name>
```
